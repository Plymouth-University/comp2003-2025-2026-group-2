<h1>Log Smart</h1>

<h3>User Stories</h3>

<h4>Customer Admin User Stories</h4>

As an Admin, I want to be able to create my account and my company 

As an Admin, I should be able to create, edit, and deactivate Standard user accounts within my registered company 

As an Admin, I should be able to create new, custom logging templates for my company 

As an Admin user, I would like to be able to use various input fields (text, numbers, checkboxes, etc.) when building a template so that I can capture the correct type of data for each log. 

As an Admin user, I would like to be able to view, edit, and deactivate existing templates so that I can keep our logging forms up to date with any changes in procedure. 

As an Admin user, I would like to be able to use pre-loaded, regulatory standard templates so that I can quickly deploy common compliance logs without building them from scratch. 

As an Admin user, I would like to be able to view, search, and filter all log entries submitted by my staff so that I can monitor operations and review records for compliance. 

As an Admin user, I would like to be able to filter logs by date range, user, and template type so that I can find specific records quickly for review or audits. 

As an Admin user, I would like to be able to export logs into an "audit-ready" format (like PDF or CSV) so that I can easily provide records to auditors or managers. 

As an Admin user, I would like to be able to view and export time & attendance reports so that I can manage payroll and track staff hours. 


<h4>Customer Standard User Stories</h4>

As a Standard user, I would like to be able to view and select from a list of active log templates so that I can find the correct form I need to fill out. 

As a Standard user, I would like to be able to fill out and submit a log so that I can record my daily tasks and compliance checks (e.g., "Fridge Temperature Check"). 

As a Standard user, I would like to be able to view a history of my own submitted logs so that I can reference my past work or confirm I completed a task. 

As a Standard user, I would like to be able to "Clock In" at the start of my shift so that my work hours are accurately recorded. 

As a Standard user, I would like to be able to "Clock Out" at the end of my shift so that my work hours are accurately recorded. 

 
<h4>General User Stories </h4>

As a user, I would like to be able to log in with a secure username and password so that I can access my account. 

As a user, I would like to be able to reset my password if I forget it so that I can regain access to my account.
